Product Description


The ASRock Radeon RX 7900 XT is a high-performance graphics card designed for gamers and content creators. It features advanced RDNA 3 architecture, delivering exceptional performance and efficiency. With 12GB of GDDR6 memory and a 192-bit memory interface, it ensures smooth gameplay and fast rendering times. The card supports the latest technologies, including DirectX 12 Ultimate and PCIe 4.0, providing an immersive gaming experience. Equipped with multiple output options like HDMI 2.1 and DisplayPort 1.4a, it allows for versatile connectivity to various displays. The dual-fan cooling system keeps the GPU temperature in check, ensuring optimal performance during intense gaming sessions. With a recommended PSU of 650W or higher, the ASRock Radeon RX 7900 XT is a powerful choice for building a cutting-edge gaming rig.
